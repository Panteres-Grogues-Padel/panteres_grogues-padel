# Panteres Grogues Pàdel 🎾

App PWA para la gestión de la sección de pàdel del colectivo LGTBQ+ Panteres Grogues.

## Funcionalidades

- 📋 Gestión de listas de jugadores por slot y club
- 🌐 Generación automática de partidos por ranking
- 📊 Ranking con eficacia y penalización por pocos partidos
- 📅 Agenda de eventos con inscripción y gestión de pagos
- 📋 Resultados con validación entre jugadores
- 🔔 Notificaciones in-app
- 🔐 Privacidad RGPD — nombre + iniciales de apellidos

## Stack

- Frontend: React (PWA)
- Backend: Supabase (PostgreSQL + Auth)
- Deploy: Vercel
